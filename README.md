# Proyecto-intermodular-Sergio-Hugo
Página web con catálogo sin css sobre una agencia de viajes low-cost.
<br><br>
***PROGRAMACIÓN DEL TRABAJO***<br><br>

**16/03 --> 22/03  Pensamos sobre qué podriamos hacer el trabajo y nos empezamos a informar y investigar páginas web de otras compañías de viajes.**<br><br>

**23/03 --> 29/03  Iniciamos con Stitch para poder partir de una base sólida para hacer el proyecto, tambien comenzamos a crear la página de inicio.**<br><br>

**30/03 --> 05/04 Creamos las páginas del catálogo, consejos y formularios de registro e inicio de sesión.**<br><br>

**06/04 --> 12/04 Buscamos las fotos que emplearemos en el proyecto, a parte de crear el pié de página y las subpáginas del catálogo.**<br><br>

**13/04 --> 19/04 Después de investigar en otras páginas web añadimos una opción al menú llamada check-in online, empezamos la documentación y usamos antigravity para mejorar el css y darle un mejor aspecto a la página web, a parte de hacerla mucho más accesible y sencilla al usuario final.**<br><br>

**20/04 --> 26/04 Terminamos la documentación, hacemos un filtrado del código para que no haya ninguna regla de css o etiqueta html que estorbe o que sea inútil y por ultimo realizamos el video opcional para una subida de nota.**<br><br>
